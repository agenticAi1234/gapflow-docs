The **Mistral** tool exposes parsing utilities used by workflows that need to normalize or extract structure from text in line with the **Mistral** integration path in the product (see `apps/sim/tools/mistral/`).

## Tool ID

| Tool ID | Purpose |
| --- | --- |
| `mistral_parser` | Parse or structure content according to the tool configuration |

Pair with [OpenAI](./openai) / model settings in **Settings** when your deployment routes parsing through Mistral-compatible endpoints.
